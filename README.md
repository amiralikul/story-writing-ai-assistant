
## Application description

its a writing assistant to create stories on different genres
It will work like co-pilot for coding. 

User starts typing begging of the story ( e.g. one sentence )
AI suggests completion ( grey text ) which user can accept by clicking tab. 

LLM will be used is GPT-4o-mini. 
UI is simple. Similar to Google sheets but much simpler. 
It should be pure Frontend application. 



# React + TypeScript + Vite + Shadcn ui + Tanstack-query

```bash
npm install
# or
yarn install
# or
pnpm install
```

2. Start the development server:
```bash
npm run dev

yarn dev

```

3. Open your browser and navigate to `http://localhost:5173`
